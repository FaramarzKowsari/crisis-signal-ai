# Academic provenance

This directory preserves the original University of Colorado Boulder deep-learning assignment and its Kaggle-score screenshot.

The notebook reflects the environment and practices used at the time. It is intentionally not treated as the production package. Historical outputs are retained for provenance, while current implementations live under `src/crisis_signal/`.

Reported notebook metrics:

- Accuracy: 0.7570584373
- Recall: 0.6995377504
- Precision: 0.7217806041

These numbers have not yet been reproduced under the modern pipeline.
