## Companion book

The related companion publication is:

**Engineering Trustworthy Crisis Intelligence: From Noisy Social Streams to Multilingual, Multimodal and Human-in-the-Loop Disaster AI**

- **Book DOI:** [10.5281/zenodo.21459117](https://doi.org/10.5281/zenodo.21459117)

## Relationship policy

The software and book are separate scholarly objects:

- the software DOI identifies executable source code and the archived release;
- the book DOI identifies the companion publication;
- neither DOI should be substituted for the other;
- the Zenodo records should be linked using complementary relations such as `isSupplementTo` and `isSupplementedBy`.

Future software releases should receive their own version-specific Zenodo DOI while preserving explicit links to the GitHub tag and the companion book.
